# Task Management System

## Tech Stack
- Frontend: React
- Backend: Django
- Database: SQLite / PostgreSQL

## Features
- User authentication
- Create, update, delete tasks
- Admin panel

## How to Run

### Backend
cd backend
pip install -r requirements.txt
python manage.py runserver

### Frontend
cd frontend
npm install
npm start
